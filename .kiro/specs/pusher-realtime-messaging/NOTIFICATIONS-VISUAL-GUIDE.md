### Desktop (â‰¥1024px)
- Banner: Full width with padding
- Settings: Card layout with icon on left
- Notification: System default (varies by OS)

### Tablet (768px - 1023px)
- Banner: Full width, slightly smaller padding

- Notification: Mobile OS style
